Python
N*N

Jak je reprezentovaná deska

Deska v observation["board"] je z pohledu aktuálního agenta:

 1 = moje kameny
-1 = soupeřovy kameny
 0 = prázdné pole

To je výhodné, protože agent nemusí řešit, jestli zrovna hraje černý nebo bílý. Vždy vidí svoje kameny jako 1.

Jaké máš možnosti řešení

Možnost 1: Úplně minimální agent

Tohle je nejjednodušší varianta, která splní rozhraní, ale nebude chytrá:

Python
class StudentAgent:
    def __init__(self, board_size: int, checkpoint_path: str | None = None):
        self.board_size = board_size
        self.checkpoint_path = checkpoint_path

    def select_action(self, observation: dict) -> int:
        legal_actions = observation["legal_actions"]
        return legal_actions[0]

Výhoda: snadné, téměř bez rizika chyby.
Nevýhoda: v soutěži bude pravděpodobně slabý.

Toto je spíš záchranná varianta, ne dobré finální řešení.

Možnost 2: Random agent

Agent náhodně vybere jeden legální tah:

Python
return random.choice(observation["legal_actions"])

Výhoda: velmi jednoduché, někdy se chová méně předvídatelně než legal_actions[0].
Nevýhoda: neučí se a dlouhodobě nebude silný.

Tohle je dobrý baseline, proti kterému pak porovnáš lepší agenty.

Možnost 3: Greedy / heuristický agent

Agent si projde všechny legální tahy a vybere ten, který okamžitě otočí nejvíc soupeřových kamenů.

Lepší heuristika může přidat:

+ bonus za rohy
+ bonus za hrany
+ bonus za mobilitu, tedy počet dalších možných tahů
- penalizace za špatná pole vedle rohů

Výhoda: poměrně jednoduché, nepotřebuje dlouhé trénování, může být silnější než random.
Nevýhoda: greedy strategie je krátkozraká. V Othellu často není nejlepší otočit nejvíc kamenů hned.

Tohle bych brala jako velmi rozumné první řešení.

Možnost 4: Deep Q-Learning agent

Tohle je hlavní doporučená varianta v zadání.

Agent dostane desku jako vstup a neuronová síť vrátí Q-hodnoty pro všechna políčka. Pro desku N×N může mít výstup N*N + 1 hodnot, tedy včetně pass akce.

Například pro 5×5:

vstup: 5×5 deska
výstup: 26 hodnot

Potom vybereš legální akci s nejvyšší Q-hodnotou. Nelegální akce musíš zamaskovat.

Typické části DQN:

neuronová síť
epsilon-greedy exploration
replay buffer
target network
reward za výhru/prohru/remízu
uložení modelu

Výhoda: odpovídá zadání, může se naučit silnější strategii.
Nevýhoda: složitější implementace, trénování může být nestabilní.

Tohle je dobrá varianta, pokud chceš mířit na plný počet bodů a soutěž.

Možnost 5: DQN + heuristiky

Tohle je podle mě pro tebe nejpraktičtější silnější řešení.

Použiješ DQN, ale pomůžeš mu heuristikami:

+ rohy jsou dobré
+ hrany jsou většinou dobré
+ mobilita je dobrá
+ pole vedle rohů mohou být nebezpečná
+ konečný rozdíl kamenů se počítá do rewardu

Můžeš to udělat dvěma způsoby:

  1. Heuristiku dát do rewardu při trénování.

  2. Heuristiku použít při výběru tahu společně s Q-hodnotou.

Například:

Python
score = q_value + alpha * heuristic_value

Výhoda: stabilnější než čisté DQN, lepší začátek učení.
Nevýhoda: musíš dávat pozor, aby heuristika nebyla moc dominantní.

Tohle bych ti doporučila jako nejlepší kompromis.

Možnost 6: Self-play

Agent hraje sám proti sobě nebo proti starším verzím sebe sama.

Výhoda: agent se neučí jen proti jednomu typu soupeře.
Nevýhoda: implementačně náročnější a může být nestabilní, pokud je agent na začátku úplně slabý.

nejdřív trénovat proti random/greedy agentovi
potom přidat self-play

4. Jak agent vybírá akci

V QLAgent.next_turn() je použita strategie epsilon-greedy.

Princip:

Python
if np.random.rand() < self.epsilon:
    action = np.random.randint(self.action_num)
else:
    action = np.argmax(self.Q_table[self.state, :])

To znamená:

s pravděpodobností epsilon zkus náhodnou akci
jinak vyber nejlepší známou akci podle Q-tabulky

U Othella bys to musela upravit tak, aby se vybíralo jen z legálních akcí:

Python
legal_actions = observation["legal_actions"]

if random.random() < epsilon:
    action = random.choice(legal_actions)
else:
    q_values = model(state)
    action = legal_action_with_highest_q_value

Tohle je hodně důležité, protože v Othellu nesmíš vrátit nelegální tah. Zadání říká, že select_action(observation) musí vrátit akci patřící do observation["legal_actions"].

5. Jak probíhá Q-learning update

V tabulkovém Q-learningu notebook používá klasickou aktualizaci:

Python
dQ = alpha * (
    reward
    + gamma * np.max(self.Q_table[new_state, :])
    - self.Q_table[old_state, action]
)

self.Q_table[old_state, action] += dQ

Význam:

nová hodnota akce se posune směrem k:
okamžitá odměna + nejlepší očekávaná budoucí odměna

U Othella by reward mohl být například:

+1 za výhru
-1 za prohru
0 za remízu
nebo rozdíl počtu kamenů na konci hry

Zadání Othella přímo zmiňuje, že reward může být založený na výhře, prohře, remíze nebo finálním rozdílu disků.

6. Akce agenta

V Othellu jsou akce jednodušší než v Tetrisu.

Pro desku N × N:

0 až N*N - 1 = polož kámen na dané pole
N*N          = pass

Zadání ale říká, že agent musí vždy vrátit jen akci z observation["legal_actions"], jinak automaticky prohraje.

To znamená, že v select_action() musíš vždy maskovat nelegální tahy.

7. Maskování nelegálních tahů

Toto je zásadní část.

Síť ti vrátí Q-hodnoty pro všechny možné akce:

Python
q_values = self.q_net(state)

Ale některé akce jsou nelegální. Proto uděláš:

Python
legal_actions = observation["legal_actions"]

masked_q = np.full(self.action_size, -np.inf)
masked_q[legal_actions] = q_values[legal_actions]

action = int(np.argmax(masked_q))

Tak agent nikdy nevybere nelegální tah.

To je rozdíl oproti Tetrisu, kde se nelegální akce často penalizovala. U Othella je lepší ji rovnou zakázat, protože nelegální tah znamená automatickou prohru.

8. Návrh neuronové sítě

Pro jednoduchou verzi bych použila obyčejnou MLP, podobně jako v Tetris DQN:

Python
class QNetwork(nn.Module):
    def __init__(self, board_size):
        super().__init__()
        input_size = board_size * board_size
        output_size = board_size * board_size + 1

        self.net = nn.Sequential(
            nn.Linear(input_size, 128),
            nn.ReLU(),
            nn.Linear(128, 128),
            nn.ReLU(),
            nn.Linear(128, output_size)
        )

    def forward(self, x):
        return self.net(x)

Tohle je nejjednodušší stabilní varianta.

Později se dá zlepšit na CNN, protože Othello je desková hra a prostorové vztahy jsou důležité. Ale pro první funkční řešení bych začala MLP.

11. Replay buffer

Převzala bych princip z Tetris DQN:

Python
transition = {
    "state": state,
    "action": action,
    "reward": reward,
    "next_state": next_state,
    "done": done,
    "next_legal_actions": next_legal_actions

U Othella je důležité ukládat i next_legal_actions, protože při výpočtu targetu chceš maximum jen přes legální akce:

Ne přes všechny akce.

12. Finální StudentAgent

Finální agent musí být velmi čistý. Žádné trénování, žádné dlouhé printy, žádný input.

Návrh:

Python
class StudentAgent:
    def __init__(self, board_size: int, checkpoint_path: str | None = None):
        self.board_size = board_size
        self.action_size = board_size * board_size + 1
        self.q_net = QNetwork(board_size)

        if checkpoint_path is not None:
            self.q_net.load_state_dict(torch.load(checkpoint_path, map_location="cpu"))

        self.q_net.eval()

    def encode_state(self, observation: dict):
        board = observation["board"]
        state = torch.tensor(board.flatten(), dtype=torch.float32)
        return state.unsqueeze(0)

    def select_action(self, observation: dict) -> int:
        legal_actions = observation["legal_actions"]

        if len(legal_actions) == 0:
            return self.board_size * self.board_size

        state = self.encode_state(observation)

        with torch.no_grad():
            q_values = self.q_net(state).squeeze(0).numpy()

        masked_q = np.full(self.action_size, -np.inf)
        masked_q[legal_actions] = q_values[legal_actions]

        action = int(np.argmax(masked_q))

        if action not in legal_actions:
            action = legal_actions[0]

        return action

Ta poslední pojistka:

Python
if action not in legal_actions:
    action = legal_actions[0]

je dobrá, protože chrání před případnou numerickou nebo indexovací chybou.

13. Heuristický fallback

Aby agent nebyl úplně závislý na síti, můžeš přidat jednoduchý fallback nebo kombinaci Q-hodnot s heuristikou.

Například:

Python
score = q_value + alpha * heuristic_score

Heuristika může dávat:

+ velký bonus za roh
+ menší bonus za hranu
- penalizace za pole vedle rohu
+ bonus za větší počet legálních tahů po tahu

Pro 5×5 jsou rohy:

Python
corners = [0, board_size - 1,
           board_size * (board_size - 1),
           board_size * board_size - 1]

Tahle hybridní strategie je v zadání také zmíněná jako možnost: DQN může být veden heuristikami, například preferováním rohů, vyhýbáním se nebezpečným pozicím nebo oceňováním mobility.

import pygame
import sys

from environment.othello_env import OthelloEnv


class OthelloGUI:
    def __init__(self, board_size=5, cell_size=90):
        pygame.init()

        self.env = OthelloEnv(board_size=board_size)

        self.board_size = board_size
        self.cell_size = cell_size
        self.margin_top = 80
        self.window_width = board_size * cell_size
        self.window_height = board_size * cell_size + self.margin_top

        self.screen = pygame.display.set_mode(
            (self.window_width, self.window_height)
        )

        pygame.display.set_caption("Othello / Reversi")

        self.font = pygame.font.SysFont(None, 32)
        self.small_font = pygame.font.SysFont(None, 24)

        self.obs = self.env.reset()
        self.done = False
        self.last_info = None
        self.last_reward = 0

    def run(self):
        clock = pygame.time.Clock()

        while True:
            self.handle_events()
            self.draw()
            pygame.display.flip()
            clock.tick(30)

    def handle_events(self):
        for event in pygame.event.get():
            if event.type == pygame.QUIT:
                pygame.quit()
                sys.exit()

            if event.type == pygame.KEYDOWN:
                if event.key == pygame.K_r:
                    self.reset_game()

                if event.key == pygame.K_p:
                    self.try_pass()

            if event.type == pygame.MOUSEBUTTONDOWN and event.button == 1:
                self.handle_click(event.pos)

    def handle_click(self, pos):
        if self.done:
            return

        x, y = pos

        if y < self.margin_top:
            return

        col = x // self.cell_size
        row = (y - self.margin_top) // self.cell_size

        if not (0 <= row < self.board_size and 0 <= col < self.board_size):
            return

        action = row * self.board_size + col

        if action not in self.obs["legal_actions"]:
            return

        self.obs, self.last_reward, self.done, self.last_info = self.env.step(action)

    def try_pass(self):
        if self.done:
            return

        pass_action = self.obs["pass_action"]

        if pass_action in self.obs["legal_actions"]:
            self.obs, self.last_reward, self.done, self.last_info = self.env.step(pass_action)

    def reset_game(self):
        self.obs = self.env.reset()
        self.done = False
        self.last_info = None
        self.last_reward = 0

    def draw(self):
        self.screen.fill((240, 240, 240))
        self.draw_status()
        self.draw_board()
        self.draw_pieces()
        self.draw_legal_moves()

    def draw_status(self):
        if self.done:
            winner = self.last_info["winner"] if self.last_info else self.env.get_winner()

            if winner == 1:
                text = "Game over: X won"
            elif winner == -1:
                text = "Game over: O won"
            else:
                text = "Game over: draw"

            restart_text = "Press R to restart"

        else:
            player = "X" if self.env.current_player == 1 else "O"
            text = f"Current player: {player}"
            restart_text = "Click a legal move | P = pass | R = restart"

        label = self.font.render(text, True, (20, 20, 20))
        label2 = self.small_font.render(restart_text, True, (70, 70, 70))

        self.screen.blit(label, (15, 15))
        self.screen.blit(label2, (15, 45))

    def draw_board(self):
        for row in range(self.board_size):
            for col in range(self.board_size):
                x = col * self.cell_size
                y = self.margin_top + row * self.cell_size

                pygame.draw.rect(
                    self.screen,
                    (34, 139, 34),
                    (x, y, self.cell_size, self.cell_size)
                )

                pygame.draw.rect(
                    self.screen,
                    (0, 80, 0),
                    (x, y, self.cell_size, self.cell_size),
                    2
                )

    def draw_pieces(self):
        board = self.env.board_abs

        for row in range(self.board_size):
            for col in range(self.board_size):
                value = board[row, col]

                if value == 0:
                    continue

                center_x = col * self.cell_size + self.cell_size // 2
                center_y = self.margin_top + row * self.cell_size + self.cell_size // 2
                radius = self.cell_size // 3

                if value == 1:
                    color = (20, 20, 20)
                else:
                    color = (245, 245, 245)

                pygame.draw.circle(self.screen, color, (center_x, center_y), radius)
                pygame.draw.circle(self.screen, (0, 0, 0), (center_x, center_y), radius, 2)

    def draw_legal_moves(self):
        if self.done:
            return

        legal_actions = self.obs["legal_actions"]
        pass_action = self.obs["pass_action"]

        for action in legal_actions:
            if action == pass_action:
                continue

            row = action // self.board_size
            col = action % self.board_size

            center_x = col * self.cell_size + self.cell_size // 2
            center_y = self.margin_top + row * self.cell_size + self.cell_size // 2

            pygame.draw.circle(
                self.screen,
                (255, 215, 0),
                (center_x, center_y),
                10
            )
